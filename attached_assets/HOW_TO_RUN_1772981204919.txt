AtomicPay — How to Run
========================

Step 1 — Open 4 terminals

  Terminal 1:   python bank_a_server.py
  Terminal 2:   python bank_b_server.py
  Terminal 3:   python gateway_server.py
  Terminal 4:   python AtomicPay.py

Step 2 — Browser opens automatically at http://localhost:7000

Step 3 — Use the App

  Create Account  →  Enter name, mobile, PIN
  Link Bank       →  Select bank, enter account number
  Send Money      →  Find recipient by mobile, enter amount
  Receive         →  See balance update, see history

Architecture:
  AtomicPay.py        ← Standalone App  (port 7000)
       ↓
  gateway_server.py   ← Payment Engine  (port 5000)
       ↓           ↓
  bank_a_server   bank_b_server
  (port 6001)     (port 6002)
